        ans = min(ans, s);
        seen.insert(s);
    }    

    string rotate(string s, int x) {
        reverse(s.begin(), s.end());

Leetcode 1638: Count Substrings That Differ by One Character

Code:

class Solution {
public:
    int countSubstrings(string s, string t) {
        int res = 0;
        for(int i = 0; i < s.size(); i++)
            res += helper(s, t, i, 0);
        for(int j = 1; j < t.size(); j++)
            res += helper(s, t, 0, j);
        return res;
    }

    int helper(string s, string t, int i, int j) {
        int res = 0, pre = 0, cur = 0;
        for(int n = s.size(), m = t.size(); i < n && j < m; i++, j++) {
            cur++;
            if(s[i] != t[j]) {
                pre = cur;
                cur = 0;
            }
            res += pre;
        }
        return res;
    }
};

Problem Statement

The problem requires counting the number of substrings in two given strings, (s) and (t), such that the two substrings differ by exactly one character. A substring is defined as a contiguous sequence of characters within a string. This problem tests the understanding of substring manipulation and string comparison in programming.

Leetcode 1863: Sum of All Subset XOR Totals

Code:

class Solution {
public:
    int subsetXORSum(vector<int>& nums) {
        int res = 0;
        for (auto i = 1; i < (1 << nums.size()); ++i) {
            int total = 0;
            for (auto j = 0; j < nums.size(); ++j)
                if (i & (1 << j))
                    total ^= nums[j];
            res += total;
        }
        return res;
    }
};

Problem Statement

The problem is to calculate the sum of the XOR values of all possible subsets of a given integer array. In this context, the XOR operation is a bitwise operation that takes two bits and returns 1 if they are different and 0 if they are the same. Given an array of integers, we need to find all possible subsets, compute the XOR of the elements in each subset, and return the total sum of these XOR values.

Leetcode 1925: Count Square Sum Triples

Code:

class Solution {
public:
    int countTriples(int n) {
    vector<bool> squares(n * n + 1);
    for (int i = 1; i <= n; ++i)
        squares[i * i] = true;
    int res = 0;
    for (int i = 1; i <= n; ++i)
        for (int j = i; i * i + j * j <= n * n; ++j)
            res += squares[i * i + j * j] * 2;
    return res;
}
};

Problem Statement

The problem involves counting the number of unique triples ((a, b, c)) where (a), (b), and (c) are integers ranging from (1) to (n) such that (a^2 + b^2 = c^2). In other words, we need to identify the Pythagorean triples formed by these integers within the given range.
