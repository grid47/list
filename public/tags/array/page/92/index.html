Leetcode 2148: Count Elements With Strictly Smaller and Greater Elements

Code:

class Solution {
public:
    int countElements(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        int res = 0, l = -1, r = -1, n = nums.size();
        for(int i = 1; i < n; i++) {
            if(nums[i] > nums[i - 1]) {
              l = i;
              break;
            }
        }
        for(int i = n - 1; i > 0; i--) {
            if(nums[i] > nums[i - 1]) {
              r = i;
              break;
            }
        }
        if(l < r) return r - l;
        return 0;
    }
};

Problem Statement

The task is to count the number of elements in an array that are strictly between the minimum and maximum values of that array. In other words, we need to identify all elements that are greater than the minimum element and less than the maximum element. If there are no such elements, the function should return 0.

Leetcode 2149: Rearrange Array Elements by Sign

Code:

class Solution {
public:
    vector<int> rearrangeArray(vector<int>& nums) {
        vector<int> ans(nums.size(), 0);
        int idxpos = 0, idxneg = 1;
        for(int num: nums) {
            if(num > 0) {
                ans[idxpos] = num;
                idxpos += 2;
            }
            if(num < 0) {
                ans[idxneg] = num;
                idxneg += 2;
            }
        }
        return ans;
    }
};

Problem Statement

The problem at hand involves rearranging the elements of an array such that positive and negative integers are placed alternately. The rearranged array must maintain the relative order of the positive and negative numbers, meaning that within their respective categories, the order of numbers should be preserved. The key challenge here is to ensure that no two positive or negative numbers appear consecutively. If this is not possible, we return an array where positive and negative integers are still rearranged, but the ordering constraint may not hold.

        int h[1001]={};

        int low = 0, same = 0, high;
        for(auto num: nums) {
            if(num < pivot) low++;
            else if(num == pivot) same++;

        high = low + same;
        same = low;
        low = 0;
